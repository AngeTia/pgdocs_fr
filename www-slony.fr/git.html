Accès au dépôt Git

Le dépôt CVS slony1-engine est maintenant obsolète, les prochains développements se feront via un dépôt Git.

Le dépôt Git officiel est disponible sur git.postgresql.org ; voir notamment la page slony1-engine.git pour plus de détails.

Il existe différents moyens permettant d'accéder au dépôt, ces moyens dépendants des différents protocoles gérés :

  • git://git.postgresql.org/git/slony1-engine.git
  • http://git.postgresql.org/git/slony1-engine.git
  • ssh://git@git.postgresql.org/slony1-engine.git

Il y a de fortes chances qu'il existe aussi un clone sur GitHub.com.
